| Роль                                                          | Права роли                                            | Группы пользователей                                                             |
|---------------------------------------------------------------|-------------------------------------------------------|----------------------------------------------------------------------------------|
| cluster-admins                                                | Полный доступ ко всему кластеру (все неймспейсы)      | Администраторы кластера                                                          |
| secrets-managers                                              | Чтение/запись секретов в отдельном неймспейсе         | Привилегированная группа для управления секретами                                |
| secrets-viewers                                               | Чтение секретов в отдельном неймспейсе                | Привилегированная группа для просмотра секретов                                  |
| domain-viewers                                                | Просмотр ресурсов домена (например, ЖКУ или Клиентов) | Операторы группы сервисов                                                        |
| Название роли, которое отвечает требованиям RBAC в Kubernets. | Укажите права, которые необходимо выдать этой роли.   | Выделите группы пользователей в организации, которые нужно связать с этой ролью. |
